Academic Handbook Current Edition COMP 2021 Software Engineering Fundamentals Credit Points 10 Legacy Code 301343 Coordinator Paul Davies Opens in new window Description This subject introduces software engineering principles including basic software lifecycle concepts, modern development methodologies and version control, conceptual modelling and how these activities relate to programming. Students apply this knowledge together with essential team-based project management to design, build and deploy a simple web-based application. The subject provides students with the skills required in software design and development and prepares them to work in a team setting.

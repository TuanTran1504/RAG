Academic Handbook Current Edition COMP 3014 Networked Systems Design Credit Points 10 Legacy Code 300575 Coordinator Sharon Griffith Opens in new window Description This subject builds on and consolidates the skills and knowledge gained in  Computer Networking and Computer Networks and Internets. Students successfully completing this subject will acquire the necessary design skills and knowledge required to build and configure enterprise scale networks. The subject provides students with an opportunity to develop problem-solving techniques and decision-making skills to resolve networking issues. Students completing this subject and its prerequisites should also now be prepared to attempt world recognized network industry certification (CCNA). Level Undergraduate Level 3 subject Pre-requisite(s) COMP 3007 Equivalent Subjects LGYA 5741 - Broadband Networking Learning Outcomes On successful completion of this subject, students should be able to: Design the physical layout of the network and produce a specification diagram
Provide detailed network documentation in the appropriate format
Design an Internet Protocol (IP) subnetting scheme using Variable Length Subnet Mask (VLSM) & Classless Inter-Domain Routing (CIDR)
Design and propose appropriate router and switch configurations
Verify and troubleshoot various types of network connections
Provide a written final report/network design proposal to a professional standard
Correctly configure a Wireless Local Area Network (WLAN)
Correctly configure single-area Open Shortest Path First (OSPF) protocol
Correctly configure Virtual Local Area Networks (VLANs) and 802.1q trunking
Correctly configure Frame Relay
Correctly configure Dynamic Host Configuration Protocol (DHCP)
Correctly configure Network Address Translation (NAT)
Create and apply Access Control Lists (ACLs)

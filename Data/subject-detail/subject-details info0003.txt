Academic Handbook Current Edition INFO 0003 Advanced Computer Studies (WSTC Prep) Credit Points 10 Legacy Code 700259 Coordinator Buddhima De Silva Opens in new window Description This subject has been developed to enhance a student's problem solving skills in the context of software development. Students are introduced to more advanced material in computer studies, including advanced features of Microsoft Access, web page development using HTML and CSS, the fundamentals of Systems Analysis and Design, programming and object-oriented analysis.

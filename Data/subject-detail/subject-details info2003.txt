Academic Handbook Current Edition INFO 2003 Technologies for Mobile Applications Credit Points 10 Legacy Code 300976 Coordinator Evan Crawford Opens in new window Description This subject introduces students to the technologies used to develop and deploy mobile applications. The subject covers evaluating organisational needs in the mobile space, responsive web design, web technologies, interface challenges, location awareness, cloud services and data storage.
